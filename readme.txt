A simple wordpress plugin to customize the default logo link.
 
== Description ==
 
This plugin let you customize the default wordpress blog link on the custom logo. This is useful if the blog is in the subdirectory and you want to link the logo to the main root website.
 
== Installation ==
 
1. Upload the plugin folder to your /wp-content/plugins/ folder.
2. Go to the **Plugins** page and activate the plugin.
3. At the settings page set the desired url.
